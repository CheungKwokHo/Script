## mtproxy.sh

- 脚本说明: Mtproto Proxy 一键安装管理脚本
- 系统支持: CentOS7 / Debian7+ / Ubuntu14+

#### 下载安装:
``` bash
wget -N --no-check-certificate https://raw.githubusercontent.com/CheungKwokHo/Script/master/mtproxy.sh && chmod +x mtproxy.sh && bash mtproxy.sh
```
